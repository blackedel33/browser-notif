BrowserNotif.JS

Lets a web page send notifications that are displayed outside the page at the system level

View on GitHub Download .zip Download .tar.gz

Instruction Demo Page

  1. Click here to request notification permission, click allow
  2. Wait one second and first notification will appear
  3. Switch to another app and wait about three second
  4. Second notification will appear
  5. You can click on the notification
  6. If notification did not show, check console, find the error message then create an issue
