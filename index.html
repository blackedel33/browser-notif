BrowserNotif.JS

Lets a web page or app send notifications that are displayed outside the page at the system level

View project on GitHub

Instruction Demo Page

  1. Click allow on pop-up request notification permission
  2. Wait one second and first notification will appear
  3. Switch to another app and wait about three second
  4. Second notification will appear
  5. You can click on the notification
  6. If notification did not show, check console find error message then create an issue
